# 小车控制

一款通过蓝牙低功耗（BLE）控制小车的 Android 遥控应用。

## 功能概述

- 扫描并连接附近的 BLE 小车设备
- 6 个方向控制（前进、后退、左转、右转、左平移、右平移）+ 停止
- 4 档速度调节
- 实时显示当前发送的指令码
- 深色主题界面

## 界面说明

应用只有一个页面（竖屏固定），从上到下依次为：

| 区域 | 说明 |
|------|------|
| 状态栏 | 显示连接状态（未连接/连接中/已连接）和连接/断开按钮 |
| 指令显示 | 实时显示正在发送的十六进制指令，如 `发送：0x04` |
| 方向按键区 | 3×3 九宫格方向键，中间为红色停止键 |
| 速度调节 | 滑块选择 1档 ~ 4档 |

方向键布局：

```
[左平移]  [前进]  [右平移]
[左转]    [停止]  [右转]
          [后退]
```

## 使用步骤

1. 打开应用，确保手机蓝牙已开启
2. 点击右上角 **连接设备** 按钮，授予蓝牙权限
3. 等待 3 秒扫描完成，从弹窗列表中选择目标设备
4. 连接成功后（状态变绿），方向键和速度滑块变为可用
5. 按住方向键控制小车，松开即停止（按停止键发送停止指令）
6. 拖动速度滑块调整档位
7. 点击 **断开** 可断开连接

## 通信协议

**传输层**：Bluetooth Low Energy (BLE 4.x+)

**GATT 配置**：

| 项目 | 值 |
|------|----|
| Service UUID | `0000FFF0-0000-1000-8000-00805F9B34FB` |
| Characteristic UUID | `0000FFF2-0000-1000-8000-00805F9B34FB` |
| 写入类型 | Write Without Response |
| 发送频率 | 100ms 一次（10 Hz） |

**指令编码**：发送单字节指令，编码规则为：

```
指令字节 = 方向基码 + 速度档位(0~3)
```

| 动作 | 基码 | 1档 | 2档 | 3档 | 4档 |
|------|------|-----|-----|-----|-----|
| 停止 | `0x00` | `0x00` | — | — | — |
| 前进 | `0x04` | `0x04` | `0x05` | `0x06` | `0x07` |
| 后退 | `0x08` | `0x08` | `0x09` | `0x0A` | `0x0B` |
| 左转 | `0x0B` | `0x0B` | `0x0C` | `0x0D` | `0x0E` |
| 右转 | `0x10` | `0x10` | `0x11` | `0x12` | `0x13` |
| 左平移 | `0x14` | `0x14` | `0x15` | `0x16` | `0x17` |
| 右平移 | `0x18` | `0x18` | `0x19` | `0x1A` | `0x1B` |

> 停止时速度档位强制为 0，始终发送 `0x00`。

## 构建要求

| 项目 | 要求 |
|------|------|
| 最低 Android 版本 | Android 6.0 (API 23) |
| 目标 SDK | Android 15 (API 35) |
| 开发语言 | Kotlin |
| 必须硬件 | 支持 BLE 的蓝牙模块 |

## 权限说明

| 权限 | 用途 | 适用系统 |
|------|------|----------|
| `BLUETOOTH` / `BLUETOOTH_ADMIN` | 基础蓝牙操作 | Android 11 及以下 |
| `ACCESS_FINE_LOCATION` | BLE 扫描需要定位权限 | Android 11 及以下 |
| `BLUETOOTH_SCAN` | 扫描 BLE 设备 | Android 12+ |
| `BLUETOOTH_CONNECT` | 连接 BLE 设备 | Android 12+ |

## 开发环境

- Android Studio
- AGP 9.2.0 / Gradle 9.4.1
